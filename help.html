A Help For PyCoder

Menu bar

There are five menus in the menu-bar.

File

File menu allows you to open/new/save/save-as some files.

Code

Code menu has a *run* menu. It can run the code. You can also press F5 to run the code too.

Edit

There are some hot keys/shortcut keys in the Edit menu. [Ctrl+Z]Undo
[Ctrl+X]Cut
[Ctrl+C]Copy
[Ctrl+V]Paste
[Ctrl+A]Select All
